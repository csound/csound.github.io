arduinoStop

arduinoStop — Ferme un port série utilisant le protocole Arduino.

Description

Opcode du greffon serial.

Ferme un port série pour arduino avec utilisation du protocole Csound-Arduino.

Syntaxe

 arduinoStop iPort

Initialisation

iPort -- Numéro de port obtenu d'un opcode arduinoStart.
