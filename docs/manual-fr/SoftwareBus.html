Bus logiciel

Csound implémente un bus logiciel pour le routage interne ou le routage vers des logiciels externes en appelant l'API de Csound.

Les opcodes pour utiliser le bus logiciel sont :

  • chn_k

  • chn_a

  • chn_S

  • chnclear

  • chnexport

  • chnget

  • chnmix

  • chnparams

  • chnset
