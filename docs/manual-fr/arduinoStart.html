arduinoStart

arduinoStart — Ouvre un port série pour l'utiliser avec le protocole Arduino.

Description

Opcode du greffon serial.

Ouvre un port série pour arduino avec utilisation du protocole Csound-Arduino.

Syntaxe

iPort arduinoStart SPortName [, ibaudRate]

Initialisation

SPortName -- Numéro nom du port

ibaudrate -- Débit série, par défaut 9600 bps.
