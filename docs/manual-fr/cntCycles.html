cntCycles

cntCycles — Retourne le nombre de fois qu'un compteur a bouclé.

Description

Opcode du greffon counter.

Retourne le nombre de fois qu'un compteur a bouclé.

Syntaxe

kval cntCycles icnt

Initialisation

icnt -- l'identifiant d'un objet compteur obtenu d'un appel à cntCreate.

Exécution

Compte le nombre de fois qu'un compteur a été redémarré.

kval -- valeur retournée.

Voir aussi

cntCreate count count_i
