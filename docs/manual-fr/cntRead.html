cntRead

cntRead — Lit la valeur courante d'un objet compteur.

Description

Opcode du greffon counter.

Lit la valeur courante d'un objet compteur sans la changer.

Syntaxe

kval cntReadicnt

Initialisation

icnt -- un identifiant pour le compteur.

`

Exécution

Lit un objet compteur.
