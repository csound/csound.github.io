cntDelete

cntDelete — Efface un compteur.

Description

Opcode du greffon counter.

Efface un compteur et libère la mémoire utilisée.

Syntaxe

kval cntDelete icnt

Initialisation

icnt -- un identifiant pour le compteur obtenu d'un appel à cntCreate.

Exécution

kval -- l'identifiant effacé ou un nombre négatif si ce compteur n'existait pas.
