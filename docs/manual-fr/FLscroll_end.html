FLscroll_end

FLscroll_end — Un opcode FLTK qui marque la fin d'une zone avec barres d'ascenseur.

Description

Opcode du greffon widgets. Ce greffon se trouve dans le répertoire des greffons et il doit être installé séparément. Les widgets FLTK ne fonctionnent pas sur Mac. Le répertoire des greffons est ici : https://github.com/csound/plugins

Un opcode FLTK qui marque la fin d'une zone avec barres d'ascenseur. C'est un autre nom pour FLscrollEnd fourni pour des raisons de compatibilité. Voir FLscrollEnd.
