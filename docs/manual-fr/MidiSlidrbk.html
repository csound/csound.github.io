Banques de réglettes

Les opcodes pour les bancs de réglettes de contrôleurs MIDI sont :

  • slider8

  • slider8f

  • slider16

  • slider16f

  • slider32

  • slider32f

  • slider64

  • slider64f

  • s16b14

  • s32b14

  • sliderKawai

Les opcodes pour stocker des bancs de réglettes de contrôleurs MIDI dans des tables sont :

  • slider8table

  • slider8tablef

  • slider16table

  • slider16tablef

  • slider32table

  • slider32tablef

  • slider64table

  • slider64tablef
