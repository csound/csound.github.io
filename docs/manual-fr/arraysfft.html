Opcodes spectraux basés sur des tableaux

Note

Ces opcodes sont conçus pour travailler avec des tableaux de taux-k pour la manipulation de données spectrales.

  • fft,
  • fftinv,
  • rfft,
  • rifft,
  • pvs2array,
  • pvsfromarray,
  • cmplxprod,
  • rect2pol,
  • pol2rect,
  • window,
  • r2c,
  • c2r,
  • mags et
  • phs.
