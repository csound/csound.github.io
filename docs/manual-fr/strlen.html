strlen

strlen — Retourne la longueur d'une chaîne de caractères.

Description

Retourne la longueur d'une chaîne, ou zéro si elle est vide. strlen ne s'exécute que pendant l'initialisation.

Syntaxe

ilen strlen Sstr
