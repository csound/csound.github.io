Réverbération

Les opcodes qu'on peut utiliser pour la réverbération sont :

  • alpass

  • babo

  • comb

  • freeverb

  • nestedap

  • nreverb (aussi appelé reverb2)

  • reverb

  • reverbsc

  • valpass

  • vcomb
