cntDelete_i

cntDelete_i — Efface un compteur.

Description

Opcode du greffon counter.

Efface un compteur et libère la mémoire utilisée.

Syntaxe

ival cntDelete_i icnt

Initialisation

icnt -- un identifiant pour le compteur obtenu d'un appel à cntCreate.

Exécution

ival -- l'identifiant effacé ou un nombre négatif si ce compteur n'existait pas.
